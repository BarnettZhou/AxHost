# Changelog

## 1.0.1 (2026-03-01)

### 新增功能
- 本地工具页面（AxHost Portal）上线
  - 页面入口：导航栏中的"本地工具"链接
  - 工具介绍：原型托管快捷管理客户端功能说明
  - Windows 版本下载：支持下载 AxHost Portal-Setup-1.0.2.exe
  - macOS 版本：显示"敬请期待"

### Bug 修复
- 修复下拉菜单（dropdown）无法正确展示的问题
- 修复原型操作菜单默认状态显示异常的问题
- 修复原型操作菜单点击后无法切换显示/隐藏状态的问题
- 统一下拉菜单显示/隐藏的 CSS 样式逻辑，改用 `hidden` 类控制

### 技术调整
- `base.html`: 修改 `.dropdown-menu` 的显示逻辑为 `:not(.hidden)`
- `index.html`: 修复列表视图下拉菜单样式冲突
- `admin.html`: 修复管理后台下拉菜单样式
- `index.js`: 修复操作菜单点击事件逻辑，实现显示/隐藏循环切换
- `main.py`: 添加 `/axhost-portal` 页面路由，挂载 `/uploads` 静态文件目录
- `navbar.html`: 将"本地工具"按钮改为链接形式

## 1.0.0 (2026-02-28)

首次发布

### 功能
- 用户管理：支持管理员、产品经理、开发者三种角色
- 原型管理：上传、更新、删除 Axure 原型（ZIP/HTML）
- 访问控制：公开/私密项目，密码保护
- 标签系统：为项目添加标签，支持常用标签
- 响应式 UI：卡片/列表双视图，拖拽上传
